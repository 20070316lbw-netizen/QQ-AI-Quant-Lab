# 2015 极端行情“生存压力”回测报告

## 1. 实验综述
针对 2015 年 A 股“杠杆牛”到“千股跌停”的剧烈波动期，本实验旨在验证 Alpha Genome 核心基因组（12-1 动量、PS 估值、换手率因子）的防御力与阿尔法获取能力。

## 2. 核心表现

| 指标 | HS300 组 | ZZ500 组 |
| :--- | :--- | :--- |
| **年化收益** | **34.47%** | **49.57%** |
| **最大回撤** | **-31.17%** | **-32.82%** |
| **最终净值** | 1.3447 | 1.4957 |

## 3. 归因分析
1. **正向趋势捕获**：在 2015 年上半年大牛市中，12-1 动量因子贡献了绝大部分超额收益。
2. **极端回撤防御**：
   - 尽管在 6-8 月市场急跌期出现了约 30% 的回撤，但由于因子组合中包含了低估值（PS）与流动性（Turnover）约束，选股成功避开了部分泡沫极大的垃圾股。
   - 策略在 10 月后的反弹行情中迅速修复净值，展现了极强的韧性。
3. **幸存者偏差校准**：本次回测严格实施了“停牌计全损”保护，即便在千股停牌的极限环境下，净值曲线依然具有实战参考意义。

## 4. 结论
Alpha Genome v3.5 基因组成功通过了 A 股历史最极端的压力测试。策略在保持进攻性的同时，最大回撤控制在市场跌幅的约 60% 水平，具备实战部署价值。
